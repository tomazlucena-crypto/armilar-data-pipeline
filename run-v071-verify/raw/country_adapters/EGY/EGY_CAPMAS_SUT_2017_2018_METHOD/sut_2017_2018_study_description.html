Overview
Abstract
Supply and Use tables represent complete framework for resources and uses within National Accounts System framework. Supply tables show displaying products in national economy sources (local or imported) ,Use tables also show how these resources are used from the products, as well as the other primary incomes generated by the production process, where the added value can be derived from it.
